# Command Line Weather Report

Author: Chengran Li

Date: 20200123

## installation

```zsh
cd to/the/path/of/your/directory
chmod a+x install.sh
sh install.sh
```

**Note**

- Make sure python 3.8 is installed and the pip is linked with python. 

- Only works for MacOS with zsh shell

- Please unzip the data file in the current directory


## Run

```zsh
weather
```
